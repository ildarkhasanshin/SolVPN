## 💡 How To Use

1. Copy a subscription URL  
2. Open your VPN client (v2rayNG, V2RayN, Shadowrocket, Clash, Kitsunebi, Trojan Client)  
3. Import the subscription link  
4. Enable auto-update if supported  
5. Done ✅

## 🔁 Update Schedule

The **entire repository** (Protocols, Countries, Subscriptions, all_configs.txt) is **automatically updated every 1 hour**.

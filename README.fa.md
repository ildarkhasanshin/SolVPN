## ❓ V2Ray چیست؟

V2Ray یک ابزار پیشرفته برای دور زدن محدودیت‌های اینترنتی و ایجاد اتصال امن است.  
از پروتکل‌هایی مثل **VMESS، VLESS، Shadowsocks و Trojan** پشتیبانی می‌کند.
